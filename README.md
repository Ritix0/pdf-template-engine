# PDF Template Engine

Универсальный инструмент на Python для автоматического заполнения PDF-шаблонов данными.

## Особенности
- Генерация QR-кодов.
- Динамическая загрузка шаблонов (координаты в .py файлах).
- Умное форматирование дат и денежных сумм.
- Современный интерфейс на CustomTkinter.

## Установка
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите `gui.py`.
